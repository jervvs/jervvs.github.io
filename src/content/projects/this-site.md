This site is built with [Astro](https://astro.build), a framework that turns simple Markdown files into a fast, polished website. The entire design philosophy is: **define the structure once, then just write.**

Every piece of content — posts, photos, projects — is a plain text file with some metadata at the top. The site validates it, styles it, and publishes it automatically.

I wrote about the full approach, the design decisions, and why I think more people should build their own: [How I Built This Site (And Why You Should Steal It)](/writing/building-a-portfolio-with-astro/).

The code is open source. Fork it, replace my content, change the colors — you can have your own version running in an afternoon.
