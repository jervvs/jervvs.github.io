A set of small, focused command-line tools for querying logs and metrics. Born out of frustration with switching between browser tabs to answer simple questions like "what happened to this service in the last 10 minutes?"

The philosophy: **one question, one command, one answer.** No dashboards, no context-switching, no waiting for a UI to load. Type a question into your terminal and get the answer.

Each tool does one thing well:
- Query log streams with filters
- Pull metric values for a time range
- Discover what labels and values exist
- Tail logs in real time

Built for people who live in the terminal and want to stay there.
